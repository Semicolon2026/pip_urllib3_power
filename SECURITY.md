# Security Policy

## Supported Versions

This is an educational project. Only the latest version is maintained.

| Version | Supported |
|--------|----------|
| 0.1.x  | ✔        |

---

## Reporting a Vulnerability

If you discover a vulnerability, please report it by opening an issue.

For serious issues, avoid public disclosure until a fix is available.

---

## Scope

This project is NOT intended for production use.
It is a learning implementation of HTTP client architecture.
